# 美食天堂 - 六大特色餐厅网站

## 项目简介

这是一个模仿公司官网风格的餐厅菜品展示网站，展示了六大特色餐厅的精选菜品及价格。

## 数据规模

- **6个特色餐厅**：翰苑、博苑、慧苑、湖苑、雅苑、楠苑
- **每个餐厅35个档口**
- **每个档口10道菜品**
- **总计：2100道菜品**

## 功能特性

### 1. 响应式设计
- 桌面端：4列网格布局
- 平板端：2列网格布局
- 移动端：单列布局 + 汉堡菜单

### 2. 餐厅筛选
- 点击导航菜单可筛选显示特定餐厅
- "全部餐厅"显示所有内容

### 3. 幸运转盘抽取 🎰
**核心功能：**
- 点击转盘中心按钮开始旋转
- 随机抽取六大餐厅之一
- 流畅的旋转动画效果（至少旋转5圈）
- 4秒动画后显示抽取结果

**两种抽取模式：**
1. **随机餐厅模式**：只抽取餐厅，不显示特写
2. **随机菜品模式**：抽取餐厅并推荐一道菜品，自动弹出特写模态框

**交互体验：**
- 旋转过程中防止重复点击
- 结果显示餐厅图标和名称
- "查看菜品"按钮跳转到对应餐厅区域

### 4. 菜品特写展示 ✨
**触发条件：**
- 选择"随机菜品"模式抽取时自动弹出
- 转盘停止后延迟500ms显示

**特写内容：**
- 🎉 庆祝动画（彩带、星星飘落效果）
- 🏪 餐厅信息（图标 + 名称）
- 🖼️ 菜品大图展示（300px高度，支持悬停放大）
- 📝 菜品名称和详细描述
- 💰 今日特惠价格（醒目的价格标签）
- 🔘 操作按钮：
  - "查看详情"：跳转到对应餐厅菜品区域
  - "再抽一次"：关闭特写并重新旋转转盘

**视觉效果：**
- 渐变色"恭喜您抽中了！"标题
- 大尺寸卡片（最大宽度700px）
- 圆角边框和阴影效果
- 图片叠加层渐变效果
- 按钮悬停动画

**关闭方式：**
- 点击右上角关闭按钮（×）
- 点击模态框外部遮罩层
- 按ESC键关闭

### 5. 菜品详情
- 点击菜品卡片或"查看详情"按钮
- 弹出模态框显示详细信息
- 支持ESC键关闭

### 6. 平滑滚动
- 点击导航自动滚动到对应区域
- Hero按钮平滑滚动到内容区

### 7. 动画效果
- 页面加载动画
- 卡片悬停效果
- 图片缩放动画
- 导航下划线动画
- 转盘旋转动画
- 特写模态框滑入动画
- 彩带飘落动画

## 使用说明

### 1. 直接打开
双击 `index.html` 文件即可在浏览器中打开网站

### 2. 添加真实图片
将菜品图片放入 `images` 文件夹，按以下命名规则命名：

**图片命名格式：**
```
{餐厅类型}-stall{档口号}-dish{菜品号}.jpg
```

**示例：**
- `chinese-stall1-dish1.jpg` - 翰苑档口1的第1道菜
- `western-stall5-dish3.jpg` - 博苑档口5的第3道菜
- `japanese-stall10-dish8.jpg` - 慧苑档口10的第8道菜
- `korean-stall20-dish5.jpg` - 湖苑档口20的第5道菜
- `dessert-stall30-dish2.jpg` - 雅苑档口30的第2道菜
- `cafe-stall35-dish10.jpg` - 楠苑档口35的第10道菜

**图片要求：**
- 格式：JPG、PNG、WebP
- 尺寸：建议 300x200 像素或更大
- 大小：建议每张不超过 500KB
- 总数：2100张图片（目前使用占位符显示）
