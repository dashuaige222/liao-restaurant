# 美食天堂 - 六大特色餐厅网站

## 项目简介

这是一个模仿公司官网风格的餐厅菜品展示网站，展示了六大特色餐厅的精选菜品及价格。网站采用现代化的前端技术栈，具有响应式设计、动态数据生成、交互式转盘抽取等丰富功能。

## 数据规模

- **6个特色餐厅**：翰苑、博苑、慧苑、湖苑、雅苑、楠苑
- **每个餐厅35个档口**
- **每个档口10道菜品**
- **总计：2100道菜品**

## 功能特性

### 1. 动态数据生成
- 使用JavaScript动态生成所有餐厅、档口和菜品数据
- 每个餐厅根据类型自动生成对应的菜品模板
- 菜品价格随机浮动，增加真实感
- 支持招牌推荐和人气爆款标签

### 2. 响应式设计
- **桌面端**：4列网格布局，完整导航菜单
- **平板端**：2列网格布局，自适应布局
- **移动端**：单列布局 + 汉堡菜单，优化触摸体验
- 使用CSS媒体查询实现断点切换

### 3. 餐厅筛选功能
- 点击导航菜单可筛选显示特定餐厅
- "全部餐厅"显示所有内容
- 筛选时带有淡入动画效果
- 自动平滑滚动到内容区域

### 4. 幸运转盘抽取 🎰
**核心功能：**
- 点击转盘中心按钮或转盘任意位置开始旋转
- 随机抽取六大餐厅之一
- 流畅的旋转动画效果（至少旋转5圈，共1800度）
- 4秒动画后显示抽取结果

**两种抽取模式：**
1. **随机餐厅模式**：只抽取餐厅，不显示特写
2. **随机菜品模式**：抽取餐厅并推荐一道菜品，自动弹出特写模态框

**交互体验：**
- 旋转过程中防止重复点击（isSpinning状态控制）
- 结果显示餐厅图标和名称
- "查看菜品"按钮跳转到对应餐厅区域
- 转盘扇区动态生成，每个扇区60度

### 5. 菜品特写展示 ✨
**触发条件：**
- 选择"随机菜品"模式抽取时自动弹出
- 转盘停止后延迟500ms显示

**特写内容：**
- 🎉 庆祝动画（彩带、星星飘落效果）
- 🏪 餐厅信息（图标 + 名称）
- 🖼️ 菜品大图展示（支持图片加载失败的占位符）
- 📝 菜品名称和详细描述
- 💰 今日特惠价格（醒目的价格标签）
- 🔘 操作按钮：
  - "查看详情"：跳转到对应餐厅菜品区域
  - "再抽一次"：关闭特写并重新旋转转盘

**视觉效果：**
- 渐变色"恭喜您抽中了！"标题
- 大尺寸卡片（最大宽度700px）
- 圆角边框和阴影效果
- 图片叠加层渐变效果
- 按钮悬停动画

**关闭方式：**
- 点击右上角关闭按钮（×）
- 点击模态框外部遮罩层
- 按ESC键关闭

### 6. 菜品详情模态框
- 点击菜品卡片或"查看详情"按钮触发
- 弹出模态框显示详细信息（图片、名称、描述、价格）
- 图片加载失败时自动使用占位符
- 支持ESC键关闭
- 点击遮罩层关闭

### 7. 平滑滚动
- 点击导航自动滚动到对应区域
- Hero按钮平滑滚动到内容区
- 考虑固定导航栏高度，计算正确的滚动位置

### 8. 导航栏效果
- 固定定位，滚动时始终可见
- 滚动超过50px时增加阴影效果
- 移动端汉堡菜单切换动画

### 9. 图片懒加载
- 使用Intersection Observer API实现
- 图片进入视口50%时才加载
- 优化页面加载性能

### 10. 页面加载动画
- 餐厅区域滚动时淡入动画
- 使用Intersection Observer监测元素进入视口
- 添加fade-in-visible类触发动画

### 11. 丰富的动画效果
- 页面加载动画
- 卡片悬停效果（阴影、缩放）
- 图片缩放动画
- 导航下划线动画
- 转盘旋转动画（CSS transform）
- 特写模态框滑入动画
- 彩带飘落动画
- 按钮悬停效果

## 使用说明

### 1. 直接打开
双击 `index.html` 文件即可在浏览器中打开网站

### 2. 添加真实图片
将菜品图片放入 `images` 文件夹，按以下命名规则命名：

**图片命名格式：**
```
{餐厅类型}-stall{档口号}-dish{菜品号}.jpg
```

**示例：**
- `chinese-stall1-dish1.jpg` - 翰苑档口1的第1道菜
- `western-stall5-dish3.jpg` - 博苑档口5的第3道菜
- `japanese-stall10-dish8.jpg` - 慧苑档口10的第8道菜
- `korean-stall20-dish5.jpg` - 湖苑档口20的第5道菜
- `dessert-stall30-dish2.jpg` - 雅苑档口30的第2道菜
- `cafe-stall35-dish10.jpg` - 楠苑档口35的第10道菜

**图片要求：**
- 格式：JPG、PNG、WebP
- 尺寸：建议 300x200 像素或更大
- 大小：建议每张不超过 500KB
- 总数：2100张图片（目前使用占位符显示）

## 技术栈

### 前端技术
- **HTML5**: 语义化标签，结构清晰，完整的注释说明
- **CSS3**: 
  - CSS变量统一管理颜色、字体、间距等样式
  - Flexbox和Grid布局实现响应式设计
  - 丰富的动画和过渡效果
  - 媒体查询实现多端适配
  - 渐变、阴影、圆角等现代样式
- **JavaScript (ES6)**:
  - DOM操作和动态内容生成
  - 事件处理和事件委托
  - Intersection Observer API实现懒加载和滚动动画
  - 模态框交互逻辑
  - 防抖函数等工具函数

### 核心API
- **Intersection Observer API**: 图片懒加载、滚动动画触发
- **DOM API**: 元素创建、查询、修改
- **Event API**: 事件监听、事件对象、事件冒泡控制
- **Window API**: 滚动控制、视口信息获取

### 设计模式
- **模块化设计**: 功能分离，代码结构清晰
- **数据驱动**: 数据与视图分离，便于维护
- **渐进增强**: 基础功能优先，增强体验渐进添加
